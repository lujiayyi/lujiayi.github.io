Selected Projects

硬核项目，以展陈语言重新组织。

01 / MULTI-AGENT CONSTELLATION

心理咨询多 Agent 系统

在大模型算法实习期间，围绕心理咨询场景参与数据工程、模型微调、多 Agent 协作与前端交互开发。系统以 Router Agent、Information Agent、Diagnosis Agent、Recommendation Agent 为核心，结合专业心理学知识库与长期/短期记忆机制，提升多轮咨询中的引导能力、上下文连续性与判定可靠性。

  • Qwen3-32B / Qwen2.5-72B
  • Dify / LangChain
  • Memory / Prompt / Frontend

02 / FOULTRACKAI

MAAR-Agent：多源增强多模态 RAG 系统

为解决复杂问答场景中的信息不足、多模态利用不充分与幻觉问题，设计 Multi-source Multi-modal Agent RAG 框架。系统以感知、问题分析、多源检索、重排序、推理生成、事实验证、自我纠错为主线，融合图像、网页与文本检索，并通过 CLIP 一致性、语义一致性与事实一致性验证提升回答可靠性。

  • Llama-3.2-11B-Vision-Instruct
  • CLIP / BGE / vLLM / FAISS
  • Agent Workflow / ReAct

03 / PHISHLLM

医疗知识库 RAG 系统

作为 Hint Lab 的主要开发者，基于 Milvus Lite、BGE-M3、BGE-Reranker-v2 与 Qwen2.5 构建面向医疗问答的知识库系统，完成文档切分、向量化存储、混合检索、重排序与生成式回答链路，并加入查询优化、反馈机制与动态自我优化设计来持续提升问答质量。

  • Milvus Lite / BM25 / BGE
  • Qwen2.5 / Streamlit
  • Hybrid Retrieval / Rerank
